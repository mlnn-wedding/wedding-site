# Папка для фотографий

Поместите сюда изображения, которые хотите использовать на сайте. Сейчас подключены временные плейсхолдеры:

- `hero-placeholder.svg` — используется в блоке героя (замените на `hero.jpg` или другое название и обновите путь в `index.html`).
- `venue-placeholder.svg` — фотография площадки.
- `dresscode-*-placeholder.svg` — примеры образов для дресс-кода.
- `gallery-*-placeholder.svg` — фотографии для слайдера галереи (настраиваются в `script.js`).

Чтобы показать собственные снимки, замените файлы с теми же именами или подправьте пути в `index.html` и `script.js`. Можно загружать `.jpg`, `.png`, `.webp` или `.svg` — главное, чтобы браузер поддерживал расширение.

Если хотите хранить оригинальные названия файлов, просто измените пути в разметке и скрипте. После замены фотографий не забудьте пересобрать или заново задеплоить сайт.
